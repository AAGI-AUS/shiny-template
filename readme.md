# Shiny Template

This Shiny app template was created to standardize the layout of shiny apps developed to support projects under
Analytics for the Australian Grains Industry (AAGI).

## Structure

```
shiny-template
├── src
│   ├── about.R
│   ├── app.R
│   ├── global.R
│   ├── helpers.R
│   ├── ui.R
│   └── utils.R
└── app.R
```
